# Prismagram
Instagram clone with Express + Prisma + GraphQL + ReactJS

## 구현 기능
 - [x] Create account
 - [x] Request Secret
 - [x] Confirm Secret (Login)
 - [x] Like / Unlike a photo
 - [x] Comment on a photo
 - [x] Search by user
 - [x] Search by location
 - [x] Follow User
 - [x] Unfollow User
 - [x] Edit my profile
 - [x] See user profile
 - [x] See MY profile
 - [x] See the full photo
 - [x] Upload a photo
 - [x] Edit the photo (Delete)
 - [x] See the feed
 - [x] Send private Message
 - [x] See chats
 - [x] See chat
 - [x] Receive Message (Realtime)
